# OpenFEC (open-fec)

The Federal Election Commission (FEC) API is a RESTful web service supporting full-text and field-specific searches on FEC data. Bulk downloads are available on the current site. Information is tied to the underlying forms by file ID and image ID. Data are updated nightly.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/open-fec/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/open-fec/refs/heads/main/apis.yml)

## APIs

### OpenFEC API

RESTful web service supporting full-text and field-specific searches on Federal Election Commission data including candidates, committees, and financial data.

- **Human URL:** [https://api.open.fec.gov/developers/](https://api.open.fec.gov/developers/)
